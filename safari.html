Click the map to open

To create a clickable image that navigates to another file or
page when clicked, you use the HTML < a > tag in combination with
the < img > tag. Start by wrapping the < img > tag inside
an < a > tag. The < a > tag, or anchor tag, is used to define
a hyperlink, and the href attribute specifies the target
URL or file path where the user will be redirected upon
clicking the image (This case: "map.html").
